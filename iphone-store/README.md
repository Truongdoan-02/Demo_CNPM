# Web Bán iPhone Demo

Trang web tĩnh demo bán iPhone (HTML/CSS/JS) đã tạo sẵn.

## Cấu trúc
- `index.html` - trang chính
- `styles.css` - giao diện
- `app.js` - logic danh sách sản phẩm, giỏ hàng

## Chạy
1. Mở `index.html` bằng trình duyệt.
2. Với server local: `Live Server` hoặc `python -m http.server`.

## Tính năng
- Hiển thị 4 model iPhone
- Thêm vào giỏ
- Tính tổng và thanh toán (alert)
- Responsive mobile
